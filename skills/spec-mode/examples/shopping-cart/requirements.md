# 购物车功能 需求定义

## 1. 功能概述

### 1.1 背景
当前 LVM 项目需要添加一个购物车功能,用于管理用户选择的编程语言版本。用户可以在浏览不同语言版本时,将感兴趣的版本添加到购物车,然后批量进行安装或切换操作。

### 1.2 目标
- 提供便捷的版本管理方式
- 支持批量操作,提高用户效率
- 提供清晰的购物车状态展示

### 1.3 范围
- **包含**:
  - 添加版本到购物车
  - 从购物车移除版本
  - 查看购物车内容
  - 批量安装购物车中的版本
  - 批量切换购物车中的版本
  - 清空购物车
- **不包含**:
  - 价格计算(本产品免费)
  - 支付功能
  - 订单管理

## 2. 用户故事

### 2.1 用户角色
- **开发者**: 需要管理多个编程语言版本的开发者

### 2.2 使用场景

**场景 1: 添加版本到购物车**
- 用户输入: 在 Python 版本列表中点击"添加到购物车"按钮
- 系统响应: 版本被添加到购物车,显示成功提示
- 预期结果: 购物车数量+1,版本出现在购物车中

**场景 2: 批量安装购物车中的版本**
- 用户输入: 打开购物车,点击"批量安装"按钮
- 系统响应: 依次安装购物车中的所有版本,显示下载进度
- 预期结果: 所有版本安装成功,购物车清空

**场景 3: 查看购物车内容**
- 用户输入: 点击侧边栏的"购物车"菜单项
- 系统响应: 显示购物车页面,列出所有已添加的版本
- 预期结果: 用户可以看到购物车中的所有版本及其状态

### 2.3 验收标准
- [ ] 用户可以添加任意版本到购物车
- [ ] 用户可以移除购物车中的版本
- [ ] 用户可以批量安装购物车中的版本
- [ ] 用户可以批量切换购物车中的版本
- [ ] 用户可以清空购物车
- [ ] 购物车数量实时更新
- [ ] 购物车内容持久化存储

## 3. 功能需求

### 3.1 功能点列表
| ID | 功能描述 | 优先级 | 复杂度 |
|----|---------|-------|--------|
| FR-01 | 添加版本到购物车 | P0 | 低 |
| FR-02 | 从购物车移除版本 | P0 | 低 |
| FR-03 | 查看购物车内容 | P0 | 中 |
| FR-04 | 批量安装购物车版本 | P0 | 高 |
| FR-05 | 批量切换购物车版本 | P1 | 高 |
| FR-06 | 清空购物车 | P1 | 低 |
| FR-07 | 购物车持久化 | P1 | 中 |

### 3.2 非功能性需求
- **性能**: 购物车操作响应时间 < 100ms
- **安全性**: 购物车数据本地存储,不上传到服务器
- **可访问性**: 支持键盘导航
- **兼容性**: 支持 Windows 10+, macOS 10.15+, Linux

## 4. UI/UX 需求

### 4.1 页面/组件结构
- **购物车页面**: `src/features/version-manager/pages/ShoppingCart/`
- **购物车侧边栏徽标**: 在侧边栏"购物车"菜单项上显示数量
- **购物车项组件**: 显示单个版本信息
- **批量操作按钮**: 安装、切换、清空

### 4.2 交互流程
```
用户浏览版本列表
    ↓
点击"添加到购物车"
    ↓
版本添加到购物车
    ↓
侧边栏购物车数量更新
    ↓
用户打开购物车页面
    ↓
查看购物车内容
    ↓
执行批量操作(安装/切换)
    ↓
完成操作,清空购物车
```

### 4.3 国际化需求
- 是否需要支持多语言: 是
- 需要的翻译 key:
  - `cart.title` - 购物车
  - `cart.addToCart` - 添加到购物车
  - `cart.removeFromCart` - 从购物车移除
  - `cart.batchInstall` - 批量安装
  - `cart.batchSwitch` - 批量切换
  - `cart.clearCart` - 清空购物车
  - `cart.empty` - 购物车为空
  - `cart.itemCount` - {count} 个项目

## 5. 数据需求

### 5.1 数据结构
```typescript
interface CartItem {
  language: string;      // 编程语言类型
  version: string;       // 版本号
  addedAt: number;       // 添加时间戳
}

interface CartState {
  items: CartItem[];     // 购物车项目列表
}
```

### 5.2 API 接口
| 接口名称 | 方法 | 描述 | 参数 | 返回值 |
|---------|------|------|------|--------|
| add_to_cart | POST | 添加版本到购物车 | `{ language: string, version: string }` | `{ success: boolean }` |
| remove_from_cart | POST | 从购物车移除版本 | `{ language: string, version: string }` | `{ success: boolean }` |
| get_cart | GET | 获取购物车内容 | - | `{ items: CartItem[] }` |
| clear_cart | POST | 清空购物车 | - | `{ success: boolean }` |

### 5.3 状态管理
- 是否需要全局状态: 是
- 需要管理的状态字段:
  - `items`: 购物车项目列表
  - `total`: 购物车项目总数
